# WP7 Testing

A test launcher and blueprint for WordPress 7.0 Beta testing.

## Files

- **index.html** — Interactive test launcher with checklists for WP 7.0 Beta features
- **wp70-comprehensive-blueprint.json** — Comprehensive blueprint for WordPress Playground

## Usage

Open the [test launcher](https://courtneyr-dev.github.io/WP7-testing/) to track testing across WordPress 7.0 Beta features, or load the blueprint JSON directly in [WordPress Playground](https://playground.wordpress.net/).
